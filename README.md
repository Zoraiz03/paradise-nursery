# Paradise Nursery

A modern houseplant shopping application built with React, Redux Toolkit, and React Router.

## Project Name
**Paradise Nursery** — Where Every Plant Finds Its Home

## Features
- Landing page with background image and company description
- Product listing with 18 houseplants across 3 categories
- Add to cart with disabled state after adding
- Shopping cart with quantity controls and delete
- Redux state management
- Responsive design

## Tech Stack
- React 18
- Redux Toolkit
- React Router DOM v6
- Vite

## Getting Started
```bash
npm install
npm run dev
```
